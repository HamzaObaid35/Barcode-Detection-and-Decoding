# Barcode Detection and Decoding System

This project presents two different implementations for **barcode detection and decoding from images** using image processing techniques.

The project is implemented **twice**:
- One version using external libraries
- One version implemented from scratch using pure Python

This comparison demonstrates both **practical efficiency** and **algorithmic understanding**.
